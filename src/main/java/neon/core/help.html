Keys

Movement keys can be selected in the options menu. Choices are the numpad keys or the nine leftmost letter keys (select the correct type of keyboard you are using). The other keys can be changed in the neon.ini file. The default values are:
Ctrl
show inventory
Esc
quit game
Space
pick up items, use doors/portals, open containers
F1
show this help
F2
toggle user interface
F3
open console
F5
save game
f
fire a ranged weapon
g
cast a spell
j
show journal
l
look around
m
show map
t
talk
u
unmount
v
sneak
When you use the look, fire, cast (with a ranged spell equiped) or talk keys, you will enter the general aiming mode. In this mode, you can move the cursor with the movement keys. Pressing the shoot, magic or talk key again with the correct target selected will then actually fire a ranged weapon, cast a spell or start a conversation. Pressing the look key will cancel the aiming mode.
