Scripting guide

The Neon engine uses JavaScript as a scripting language. Some specific functions are included to interact with the engine.

Entitities

All entities in the game (items and creatures) have a uid, getEntity(<uid>) can be used to find an entity if its uid is known. Entitities have the following functions available:
<entity>.getID()
returns the string ID of an entity.
<entity>.getUID()
returns the unique identifier of an entity.
<entity>.bounds or <entity>.getBounds()
the rectangular bounds of the entity on the map.
All changes to the position of an entity should be done through its bounds component:
<entity>.bounds.setX(<new x-coordinate>)
sets the x-coordinate of an entity.
<entity>.bounds.setY(<new y-coordinate>)
sets the y-coordinate of an entity.
<entity>.bounds.setLocation(<new x-coordinate>, <new y-coordinate>)
sets x- and y-coordinates of an entity.

Creatures

Creatures have all general entity functions available, plus the following::
<creature>.getAI()
returns the AI of a creature.
<creature>.addSpeed(<speed increase>)
adds speed to a creature.
<creature>.isMember(<faction name>)
returns whether a creature belongs to the given faction or not.
<creature>.getRank(<faction name>)
returns the creature's rank in the given faction (or null when the creature does not belong to the given faction).

The player

To edit the player, use the object PC (or player). All functions available to creatures are also available to the player. To edit the position of the player, for example, type PC.bounds.setX(<new x-coordinate>) or PC.bounds.setY(<new y-coordinate>). Functions available exclusively to the player are:
PC.setSign(<new birth sign>)
gives the player a new birth sign.

AI

The AI of a creature can be retrieved with: <creature>.getAI(). Some functions available for AI's are:
<AI>.getDisposition(<other creature>)
returns the disposition of the creature with the current AI towards another creature.
